# 一億小目標

一隻 **5–10 分鐘一局** 嘅文字模擬經營遊戲：20 歲出社會，用 40 年靠炒貨、創業同家庭選擇，喺 60 歲退休前累積至少 **$100,000,000**。

風格係惡搞卡通、香港口語、黑色幽默——唔係嚴肅理財 App。

**線上遊玩：** [https://a-small-goal.vercel.app/](https://a-small-goal.vercel.app/)

完整產品同實作規格見 `[SPEC.md](./SPEC.md)`。

## 遊戲目標

| 項目 | 說明                                                       |
| ---- | ---------------------------------------------------------- |
| 起步 | 20 歲，健康 100，倉庫 100 格                               |
| 退休 | 預設 60 歲（「發達之路」可延長）                           |
| 達標 | 退休時總資產 ≥ **$1 億**（現金 + 貨物市值 + 公司持股價值） |
| 失敗 | 健康歸零猝死、長期負債破產、或者自己選擇自殺結束           |

### 資產評級（退休／結算）

| 總資產      | 評級         |
| ----------- | ------------ |
| ≥ $1 億     | 小目標達成！ |
| ≥ $1,000 萬 | 半隻腳上岸   |
| ≥ $10 萬    | 平凡中產     |
| 再低        | 社會畜生     |

## 點玩

### 1. 標題頁

- **開始遊戲**：開一局新人生
- **簡易模式**：健康消耗減半、負面現金事件較溫和（適合第一次玩；局末「發達之路」積分會打 75 折）
- **發達之路**：用累積積分永久強化下局起步（見下方）

### 2. 投胎

開局 RNG 隨機投胎到其中一個家庭，決定起步現金（同局內唔會改）：

| 家庭   | 起步資金   | 額外                                          |
| ------ | ---------- | --------------------------------------------- |
| 死窮撚 | $45,000    | 起步名聲 +5；**首次**創業、**首次**擴倉有折扣 |
| 死中產 | $100,000   | —                                             |
| 二世祖 | $1,000,000 | —                                             |

三種家庭權重相同。

### 3. 每年流程

每個回合（一年）固定係：

1. **隨機事件** — 先睇情境，再揀一個選項（改現金／健康／名聲／某貨價格等）
2. **自由操作** — 喺市場、創業、家庭之間任意來回
3. **結束今年** — 結算公司收入／維護、家用同學費、健康消耗、低健康住院、負現金清盤
4. 年齡 +1；健康 ≤ 0 → 猝死；到退休歲 → 退休評級；否則下一年再出事件

事件未揀完之前，唔可以買賣同投資。

## 核心系統

### 市場炒貨

- 商品：薯片、維他奶、智能手機、名牌波鞋、比特幣、黃金、電動車、美股期權
- 每年價格會跟通脹同隨機波幅重算；事件可以再推高／壓低某貨
- 買入要有現金同倉庫空位；賣出要有持倉
- 高風險貨（比特幣、黃金、電動車、期權）佔更多倉庫格
- **擴倉**：每次 +50 格，費用隨升級次數指數上升

提示：價錢相對「合理價」偏平就值得入貨；偏貴就考慮出貨。倉庫唔夠就先擴倉或者出清舊貨。

### 創業帝國

- 由珍珠奶茶店一路爬到房地產集團等多檔公司
- 創業要夠現金同名聲；有機會開業失敗（部分投資會退回）
- 成功後持有 **100 股（100%）**，之後可以喺股市 **增持／減持**
- 每年收入同維護費跟持股比例計；股價每年浮動
- 沽清 100% 即退出該業務；公司亦有機會倒閉（創業後有短暫保護期）

### 家庭

- 可相親對象（例如瑪麗、珍妮、潔西卡），各有結婚成本同門檻
- 結婚即時效果（名聲、現金、健康）同每年家用／健康加成唔同
- 每年有機會生仔；子女成長期間要交學費，成年後有隨機結果

### 健康、名聲同其他

| 操作         | 作用                                         |
| ------------ | -------------------------------------------- |
| 睇醫生       | 一次過回滿健康（收費跟通脹 + 總資產抽成）    |
| 捐款         | 用錢換名聲（名聲越高越貴）                   |
| 健康         | 每年固定扣血；過低會住院罰款；歸零即猝死     |
| 免費 checkup | 每隔固定歲數有一次小量回血                   |
| 自殺         | 主動結束本局（極早自殺唔會攞到發達之路積分） |
| 負債         | 現金可以暫時負數；長期撐唔住會破產出局       |

畫面上有分頁：**市場 / 創業 / 家庭 / 日誌**，底部大掣 **結束今年**。

### 里程碑

資產衝到 100 萬、1,000 萬、5,000 萬、1 億時會有里程碑提示（例如「小目標達成」）。

## 發達之路（跨局成長）

每局結束（退休、猝死、破產等）會按結果、資產評級、活咗幾耐計 **積分**，存喺本機，永久累積。

積分可升級：

| 升級     | 效果                        |
| -------- | --------------------------- |
| 初次資金 | 每級起步現金 +$15,000       |
| 初次名聲 | 每級起步名聲 +3             |
| 初次倉庫 | 每級起步倉庫 +20 格         |
| 延長退休 | 每級退休年齡 +1（由 60 起） |

簡易模式局積分較少；未完成第一年就自殺刷投胎 = 0 分。

---

## 策略提示（唔係唯一正解）

1. **早年**：低中階貨來回炒，留意事件改價；死窮撚優先用首次折扣開檔或擴倉。
2. **中期**：夠錢就開穩定現金流公司；名聲唔夠先捐款或者靠事件／結婚。
3. **健康**：唔好只顧賺，猝死 = 未退休就完；睇醫生貴但有時係必須。
4. **家庭**：伴侶係長期 buff，亦係固定開支；生太多仔學費會食現金。
5. **後期**：高階公司 + 持股調倉 + 炒高價貨，衝一億；記住通脹同維護費會愈嚟愈大。

## 技術棧

- React 19 + TypeScript + Vite
- Tailwind CSS v4
- Zustand（UI state）
- Vitest（engine 單元測試）
- 手機優先 light-mode PWA

規則層係純 TypeScript engine；UI 只呼叫 store／engine，唔直接改遊戲數值。

## 開始開發

需要 Node.js 同 [pnpm](https://pnpm.io/)。

```bash
pnpm install
pnpm dev
```

開 [http://localhost:5173](http://localhost:5173)。

### Scripts

| Command          | 用途                  |
| ---------------- | --------------------- |
| `pnpm dev`       | 本機開發伺服器        |
| `pnpm typecheck` | TypeScript 檢查       |
| `pnpm test`      | Vitest 單元測試       |
| `pnpm build`     | 生產建置              |
| `pnpm preview`   | 預覽 production build |
| `pnpm lint`      | oxlint                |
| `pnpm format`    | Prettier 格式化       |
